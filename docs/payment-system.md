# 收款系统方案

## 选择：个人收款码（最简方案）

### 流程
```
用户付款 → 截图发微信/邮箱 → 手动开通 → 发登录链接
```

### 优点
- 0 成本
- 即时到账
- 无需资质

### 缺点
- 手动处理
- 无法自动发货
